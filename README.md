# IENAD BETANIA - RESTAURACIÓN

Aplicación web de la Iglesia Restauración en Cochabamba, Bolivia.

## Características
- Información de horarios y cultos
- Peticiones de oración
- Redes sociales y contenido multimedia
- Ubicación y contactos directos

## Tecnologías
- Streamlit
- Python 3.8+
- HTML/CSS
